# 🎨 Generative-UI-Lab

[![UI/UX](https://img.shields.io/badge/Design-UI%2FUX-orange.svg)](https://github.com/YakshanAli1/Generative-UI-Lab)
[![GenAI](https://img.shields.io/badge/Tech-Generative%20AI-red.svg)](https://github.com/YakshanAli1/Generative-UI-Lab)

Exploring the intersection of AI and Product Design. This lab contains experiments in generating design tokens and CSS components using LLMs.

## 🔥 Key Highlights
- **Prompt-to-CSS:** Generate modern UI components from natural language descriptions.
- **Design Token Generator:** Automated palette and spacing system generation.
- **Accessibility Checker:** AI-powered audit for WCAG compliance.

## 📦 Usage
Check out the `/experiments` folder for latest prototypes.
